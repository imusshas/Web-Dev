Link at the same page.

To create link at the same page you have to give an id to the tag that you want to link with. Then in the href attribute of the anchor tag from where you want to link the tag you have to give its value: a hash(#) same as the id name. For example:
<a href="#link"> Go to the section </a>
Congratulations! You have linked successfully.
